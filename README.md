# VIDE - VI IDE

## Objective

The aim of this repo is to create a set of vim and shell script that provides users with a pre-costumized IDE.

The scripts provides integration with other useful repos for AI interaction, code completion and more.

## Pre-requisites

In Debian/Ubuntu:

```shell
sudo apt install vim vim-gui-common vim-doc vim-scripts git
```

### Optional

```shell
sudo apt install cscope universal-ctags screen zsh nodejs npm
```

**Note:** exuberant-ctags can be used instead universal-ctags
